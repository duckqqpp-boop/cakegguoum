📖 Sweet Patissiere Mock API

꿈빛 파티시엘 디저트 데이터를 연동할 수 있는 가상 API 가이드 문서입니다.

GET /api/desserts

전체 디저트 목록 및 상세 데이터를 반환합니다.

GET /api/desserts/:id

특정 ID를 가진 디저트의 상세 정보를 조회합니다.

GET /api/random

전체 데이터 중 랜덤으로 디저트 1개를 추천합니다.

GET /api/search?q={query}

이름 또는 재료 키워드로 실시간 필터링 검색을 수행합니다.

GET /api/categories

등록되어 있는 디저트 카테고리 목록을 불러옵니다.
